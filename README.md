# League of Legends — Player Performance Analysis

Multivariate statistical analysis of ~80k League of Legends matches: PCA, clustering (k-means, hierarchical), logistic regression, and random forest for win prediction.

## Project structure

```
├── data/              # Match tables (CSV)
├── figures/           # Plots generated by analyse.R
├── output/            # Result tables (CSV)
├── analyse.R          # Main R analysis script
├── generate_pdf.py    # Builds rapport_final.pdf from figures + output
├── rapport.Rmd        # R Markdown report (optional knit)
├── LOL report.pdf     # Written report (PDF)
└── README.md
```

## Requirements

### R packages

```r
install.packages(c(
  "tidyverse", "FactoMineR", "factoextra", "cluster",
  "corrplot", "knitr", "gridExtra", "scales", "caret", "randomForest"
))
```

### Python (optional, for `generate_pdf.py`)

```bash
pip install reportlab
```

## How to run

1. Clone the repository and open the project folder in RStudio (or set working directory to the project root).

2. Run the full analysis (creates/updates `figures/` and `output/`):

   ```r
   source("analyse.R")
   ```

3. Optional — regenerate the structured PDF report:

   ```bash
   python generate_pdf.py
   ```

4. Optional — knit `rapport.Rmd` to HTML/PDF if you use R Markdown.

## Data

CSV files in `data/` are joined in `analyse.R` via `MatchStatsTbl`, `SummonerMatchTbl`, `MatchTbl`, `RankTbl`, and `ChampionTbl`. The script samples 15,000 rows (stratified by lane) for performance.

## Outputs

- **figures/** — 19 PNG plots (PCA, clustering, classification)
- **output/** — `stats_descriptives_lane.csv`, `acp_valeurs_propres.csv`, `profils_clusters.csv`
